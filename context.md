# Project Context: Parkinson's Symptom Tracker
- **Institution**: University of Bristol (3rd Year Undergraduate Dissertation)
- **Focus**: LLM-based interactive agent for Parkinson's symptom tracking.
- **Tech Stack**: Python, Gemini API (via WhatsApp Business API), SQLite.
- **Medical Domain**: MDS-UPDRS, PDQ-39, MoCA.
- **Guidelines**: Ensure GDPR compliance, data anonymization, and academic integrity.
